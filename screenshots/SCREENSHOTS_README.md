## How to Add Screenshots

1. Open the Streamlit app at `http://localhost:8501`
2. Use your OS screenshot tool:
   - **Windows**: `Print Screen` or `Shift + Print Screen`
   - **Mac**: `Cmd + Shift + 4`
   - **Linux**: `Print Screen` or screenshot tool
3. Save images as PNG format
4. Place them in this folder
5. Update this README with descriptions
